# movie-service
API de filmes.

## Como Executar

1. suba uma instância de MongoDB na pasta data desta API (crie-a se não existir)
2. adicione dados no seu banco conforme instruído no livro
3. git clone
4. npm install
5. copie o .env.example como .env e preencha as variáveis
6. npm start
